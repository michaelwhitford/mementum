title: symbols-are-event-types
---

Mementum symbols are event types (what happened), not memory markers (what was touched). They describe insights, shifts, decisions, mistakes, wins, patterns, and meta-observations — all of which occur in both memory and code work. Union, not exclusion. Projects extend with symbols for activities outside the memory domain (🌱 new, 🏗️ build, 🐛 fix) via extend_per_domain. The mementum set is complete for memory; code-only symbols describe activities that don't exist in the memory domain.
